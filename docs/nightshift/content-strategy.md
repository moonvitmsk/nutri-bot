## 2. Шаблоны утренних и вечерних сообщений (10)

### 2.1 Утренние (5)

**УТ-1. Мотивация (streak = 1--3 дня)**
```
Доброе утро! Ты на {streak}-м дне заботы о себе.
Маленькие шаги каждый день -- и через месяц ты не узнаешь себя.

Не забудь принять витаминки Moonvit.
Отправь фото завтрака -- посмотрим, чего не хватает.
```

**УТ-2. Привычка закрепляется (streak = 4--7 дней)**
```
{name}, утро! {streak} дней подряд -- привычка начинает формироваться.

Факт дня: {random_fact}

Витаминки приняла? Отправь "да" -- отмечу в твоём дневнике.
```

**УТ-3. Стабильность (streak = 8--14 дней)**
```
{name}, {streak} дней! Ты уже в топ-20% самых дисциплинированных пользователей.

Совет: принимай Moonvit в одно и то же время -- так организм привыкает и усваивает лучше.

Как настроение сегодня? Напиши пару слов.
```

**УТ-4. Амбассадор (streak = 15--30 дней)**
```
{streak} дней подряд, {name}! Это уже не случайность -- это стиль жизни.

Замечаешь изменения? Расскажи -- мне важна обратная связь, чтобы подстроить рекомендации.
```

**УТ-5. Легенда (streak > 30 дней)**
```
{name}, {streak} дней! Ты -- пример для всех.

На этом сроке организм уже накопил запас нутриентов. Хочешь, проведём DeepCheck и посмотрим, что изменилось?
```

### 2.2 Вечерние (5)

**ВЧ-1. Итог дня (streak = 1--3)**
```
Как прошёл день, {name}?

Отправь фото ужина или расскажи, что ела -- подведём итог по нутриентам.
Спокойной ночи скоро, а пока -- давай закроем день с пользой.
```

**ВЧ-2. Напоминание о сне (streak = 4--7)**
```
{name}, через час -- идеальное время лечь.

Совет: за 30 минут до сна убери телефон. Мелатонин вырабатывается в темноте.
Если принимаешь "Сон и расслабление" -- самое время.
```

**ВЧ-3. Рефлексия (streak = 8--14)**
```
{name}, {streak} дней ты следишь за собой. Это круто.

Вопрос вечера: какую одну вещь ты сделала сегодня для своего здоровья?
Напиши -- я запомню и учту в рекомендациях.
```

**ВЧ-4. Micro-совет (streak = 15--30)**
```
Вечерний совет: {evening_tip}

{name}, ты уже {streak} дней в системе. Завтра расскажу кое-что интересное про {next_topic}. До утра!
```

**ВЧ-5. Тёплое закрытие (streak > 30)**
```
{name}, день {streak}. Ты стабильна, как швейцарские часы.

Напомню: результаты приёма витаминов накапливаются. Через 3 месяца эффект будет максимальным.
Спокойной ночи. Завтра новый день заботы о себе.
```

## Примечания к использованию

- Переменные: `{name}`, `{streak}`, `{random_fact}`, `{evening_tip}`, `{next_topic}`, `{bot_link}`
- Факты и советы подтягиваются из nutri_settings через админку
- Шаблоны шаринга используются в inline-кнопке "Поделиться" после ключевых действий (DeepCheck, анализ фото, достижение streak milestone)
- Все тексты можно редактировать через админку без деплоя
