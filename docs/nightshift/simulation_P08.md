- Запустил онбординг: спросил имя

**Что Миша сделал:**

- Пол: "м"
- Возраст: "99"
- Рост: "220"
- Вес: "30"
- Цель: "стать котом"

**Что бот сделал хорошо:**

**Чего не хватило:**
- Бот принял абсурдные данные (99 лет, 220 рост, 30 кг) без валидации
- Цель "стать котом" принята - нет фильтра на бессмысленный ввод
- Нет защиты от тролль-онбординга
