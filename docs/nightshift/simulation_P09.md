**Город:** Москва

## День 1: Контент-разведка

**Контекст:** Менеджер Moonvit написал Алисе: "Попробуй бота, сделай обзор, дадим промокод для подписчиков". Алиса ставит бота, сразу думает как это снимать.

**Что отправила боту:**
- Скриншот записала на экран (screen recording) для будущего Reels
- Фото завтрака: авокадо-тост на цельнозерновом хлебе, яйцо пашот, руккола, помидоры черри
- Онбординг: Алиса, ж, 26, 168, 57, "поддержание формы и здоровое питание"

**Что бот ответил:**
- Завтрак: ~380 ккал, белки 18г, жиры 22г, углеводы 28г
- "Идеальный баланс макронутриентов! Авокадо + яйцо = полезные жиры + белок. Руккола добавляет железо и фолиевую кислоту"

**Что бот сделал хорошо:**
- Красивый, подробный разбор - контентопригодно
- Упоминание фолиевой кислоты - "умный" ответ, который блогер может процитировать

**Чего не хватило:**
- Нет "шарибельного" формата ответа (карточка, которую можно заскринить для Stories)
- Нет функции "поделиться результатом" с красивым оформлением
