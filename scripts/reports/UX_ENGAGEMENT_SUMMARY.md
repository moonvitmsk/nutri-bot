## Критические Product Gaps

### 1. Голосовой ввод еды (7/10 в batch 3 запросили)
**Уже реализован** через Whisper (voice.ts), но:
- Нет кнопки "Записать голосом" в меню
- Нет onboarding-подсказки о голосовых
- Пользователи не знают что можно говорить

**Fix**: добавить "🎙 Голосом" кнопку в mainMenu + упомянуть в /help

### 2. Тон бота не подходит 50+ (5/30 потеряны)
Бендер+Лебедев = сильный бренд для 22-35, но:
- Ольга (55): "грубо"
- Иван (60): "непонятно"
- Татьяна (50): "несерьёзно"
- Людмила (65): "страшно"
- Виктория (42): "непрофессионально"

**Fix**: Adaptive tone — определять по возрасту из профиля. Для 45+ более уважительный тон.

### 3. Нет activity_level в онбординге (спортсмены 6/30)
Все получают `moderate` → КБЖУ занижено для:
- Алексей (марафон): недосчёт ~500 ккал/день
- Елена (кроссфит): недосчёт ~400 ккал/день
- Фёдор (плавание): недосчёт ~600 ккал/день

**Fix**: добавить step после goal — выбор активности

### 4. Нет "срыв recovery" (эмоциональные едоки)
Николай (стресс-еда) после binge видит красные цифры → стыд → бросает бота.

**Fix**: после превышения нормы на 150%+ → мягкое сообщение "Бывает! Завтра новый день" вместо "ты съел 4000 ккал"

### 5. Нет трекера веса (цель "похудеть" — 10/30)
Самая частая цель, но нет графика веса over time.

**Fix**: /weight command + история взвешиваний

### 6. Нет shareable cards (вирусный канал)
Дарья-маркетолог: "Я бы шерила отчёт в инсту"

**Fix**: красивая карточка дня/недели для шеринга (уже есть recipe-card.ts — расширить на daily summary)
